# Maven-Toys-Data-Analysis (Interactive Dashboard using Power BI)

## Project Overview
This project presents an end-to-end sales and inventory analysis of Maven Toys using Power BI. The dashboard provides insights into revenue, profit, order trends, product performance, and store-level efficiency across 2022 and 2023. It highlights key business drivers, identifies underperforming areas, and uncovers shifts in customer demand to support data-driven decision-making.

## Project Objective
To conduct a comprehensive analysis of Maven Toys’ retail performance by examining sales, profit, and inventory data across stores and product categories. The project aims to uncover trends, identify performance gaps, and deliver actionable recommendations that support revenue growth, profitability improvement, and operational efficiency..
